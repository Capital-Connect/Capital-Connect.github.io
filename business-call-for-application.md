# Call for Applications

> Browse and apply for funding calls that are global or aligned to your business profile.
> Access Call for Applications at [app.capitalconnect.africa/business/call-campaigns](https://app.capitalconnect.africa/business/call-campaigns)

## Browse Calls for Applications
A list of available funding calls — both global opportunities and those matched to your business profile.
> Access at [app.capitalconnect.africa/business/call-campaigns](https://app.capitalconnect.africa/business/call-campaigns)

Each call card shows:

| Field | Description |
|---|---|
| **Title** | Name of the funding call or programme |
| **Status badge** | Current application status — *Draft*, *Submitted* |
| **Deadline badge** | Whether the deadline has passed |
| **Description** | A brief overview of what the call is about |
| **Deadline** | The date by which applications must be submitted |
| **View** | Opens the full campaign details page |
| **My Application** | Opens your application for that specific call |

### Application Status Labels

| Status | Description |
|---|---|
| **Draft** | Application started but not yet submitted |
| **Submitted** | Application has been submitted successfully |
| **Deadline Passed** | The call deadline has passed — new applications may not be accepted |

## Campaign Details
View the full details of a specific call for applications.

Each campaign details page shows:

| Field | Description |
|---|---|
| **Status** | Whether the call is *Open* or closed |
| **Deadline** | The application closing date |
| **Overview** | A description of the programme and what it funds |
| **Focus Areas** | The specific impact or thematic areas the call targets |
| **Sectors** | The business sectors eligible to apply |
| **Target Countries** | The countries whose businesses are eligible to apply |
| **Apply panel** | Shows your current application status and a button to view or start your application |

> If the deadline has not passed, you can apply directly from this page.
> If you have already started an application, the panel shows your current status and a **View my application** button.

## My Applications
A full list of all your call for application submissions — both draft and submitted.
> Access at [app.capitalconnect.africa/business/call-campaigns/applications](https://app.capitalconnect.africa/business/call-campaigns/applications)

Each row in the list shows:

| Column | Description |
|---|---|
| **Call for Application** | The name of the funding call |
| **Deadline** | The application closing date, with a *Deadline passed* note if applicable |
| **Status** | Your submission status — *Draft* or *Submitted* |
| **Application Stage** | The review stage your application is at — *Pending*, *Under Review*, *Shortlisted* |
| **Submitted** | The date and time you submitted or last saved the application |
| **Actions** | **View** to see application details; **Edit** to update a draft or shortlisted application |

### Application Stage Labels

| Stage | Description |
|---|---|
| **Pending** | Application submitted and awaiting review |
| **Under Review** | Application is currently being reviewed by the programme team |
| **Shortlisted** | Application has been shortlisted for further consideration |

## Application Details
View the details of a specific application.
> Access at `app.capitalconnect.africa/business/call-campaigns/applications/{id}`

Shows:
- **Campaign name and deadline**
- **Application status** (e.g. *Draft*)
- **Your answers** — The responses you submitted for each question in the application form
- **View campaign** — Links back to the full campaign details page

---

## Possible Issues
- Cannot submit application — deadline has already passed
- Application stuck on *Draft* — form has not been fully completed or submitted
- Application not appearing in My Applications — try refreshing the page

## Possible Remedies
- Check the deadline on the campaign details page before applying
- Ensure all required questions in the application form are answered before submitting
- Refresh the page if a recently submitted application is not showing
- If the issue persists, contact our office

## Next Steps
After applying or managing your applications:

1. **Track Your Application** — Monitor your application stage in My Applications
2. **Edit a Draft** — Return to a draft application and complete it before the deadline
3. **Browse More Calls** — Explore other open funding calls aligned to your business profile
4. **Go to Dashboard** — Return to the main dashboard to continue your fundraising journey
