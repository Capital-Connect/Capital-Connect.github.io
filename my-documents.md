# My Documents

> Store and manage your business investment documents in one secure place.
> Your documents are only visible to you, your advisors, and the investors you have connected with.
> Access My Documents at [app.capitalconnect.africa/business/documents](https://app.capitalconnect.africa/business/documents)

---

## Requirements
- A document file to upload (e.g. PDF, Word, Excel)
- A folder to upload the document into
- An optional description for the document

---

## Sections
- Folders
- All Files

## Folders
Documents are organized into four predefined folders:

| Folder | Description |
|---|---|
| **Business Teaser** | A short, high-level overview of your business shared with investors to spark interest |
| **Business Valuation** | Your business valuation reports and supporting documents |
| **Pitch Deck** | Your investor presentation slides |
| **Financial Model** | Your financial projections, forecasts, and models |

> Each folder shows the number of files and total storage size.
> Click the **⋮** menu on any folder to manage its contents.

---

## All Files
A flat list of all uploaded documents across all folders, sortable by:
- **File** — Document name
- **Size** — File size
- **Uploaded** — Date the document was uploaded
- **Action** — Options to view, download, or delete the file

---

## Uploading a Document
Click **Upload Document** to open the Document Upload modal.

### Document Upload Form
1. **Directory** — Select the folder to upload the document into
   > Options: *Business Teaser, Business Valuation, Pitch Deck, Financial Model*
2. **Document** — Click *Choose File* to select a file from your device
3. **Description** — Add a short description about the document (optional but recommended)
4. Click **Upload** to save the document to the selected folder

## Possible Issues
- Upload button is greyed out — a folder (Directory) has not been selected or no file has been chosen
- Document not appearing after upload — try refreshing the page
- File upload fails — file may be too large or in an unsupported format

## Possible Remedies
- Ensure both a Directory and a file are selected before clicking Upload
- Refresh the page after uploading to see the latest file list
- Try compressing the file or converting it to PDF if the upload fails
- If the issue persists, contact our office

## Next Steps
After uploading your documents:

1. **Share with Investors** — Connect with matched investors so they can access your documents
2. **Get Expert Help** — Use *Get Expert Documents Preparation* if you need professional document support
3. **Go to Investor Database** — Find and connect with investors to share your documents with
4. **Go to Dashboard** — Return to the main dashboard
