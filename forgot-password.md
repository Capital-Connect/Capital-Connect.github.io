# Forgot Password

> Reset a forgoten password on Capital Connect Africa Portal

## Requirements
- Email Address

> Step 1: Open [The Login Page](https://app.capitalconnect.africa/auth)
> Step 2: Locate the "Forgot Password" call to action and click it: This opens the [Forgot Password Page ](https://app.capitalconnect.africa/auth/forgot-password)
> Step 3: Enter the email you used to create an account and hit the "Send Reset Link" button: If this is a valid email you created the account with, an email with a password reset link will be sent here.
> Step 4: Open your and find the Reset Password email from Capital Connect Africa, check spam if it does not arrive in your inbox in 2 minutes. Whitelist if email went to spam

## Possible Issues: 
- "Unable to create a password request please try again later" Error - if the email was incorrect.
- No email in the inbox - if the user provided an invalid email or the email went to the spam box

## Possible Remedies
- User to check spam if the email was sent to it.
- User to verify that the email was correctly spelt
- If all the above check, then contact our office
